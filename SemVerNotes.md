 
 version format is Major.Minor.Patch
 Patch version will get updated  for bug fixes
 Minor version will get updated for new feature that won't break existing code/features
 Major version will get updated for new features that will break the existing code

 lodash: ^2.4.24 means interested in all version updates of Minor and patch i.e 2.x
 lodash: ~2.4.24 means interested in all version updates of Minor and patch i.e 2.4.x
 lodash: "2.4.24" means exact version is required
