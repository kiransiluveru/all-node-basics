Node.js Basics Learning

Topics Covered

  • Global Objects: Understanding globalThis and module scope
  • ES Modules: Using import/export syntax
  • Path Module: Working with file paths using path module
  • Module Properties: Accessing import.meta and module information

Key Concepts

  • import.meta.url - Current module's file URL
  • import.meta.filename - Current module's filename
  • import.meta.dirname - Current module's directory
  • globalThis - Global scope object in Node.js
  • path.parse() - Parse file paths into components

Running the Project

To run the Node.js scripts:

node index.js
node path_module.js
node global_obj.js

Project Structure

  • index.js - Main entry point
  • global_obj.js - Global objects demonstration
  • path_module.js - Path module usage
  • log_module.js - Custom logging module
  • package.json - Project metadata with ESM configuration
